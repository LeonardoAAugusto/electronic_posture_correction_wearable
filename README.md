# Electronic_Posture_Correction_Wearable
Desenvolvimento de um sistema vestível de correção postural usando Lilypad Arduino e MPU6050. Fixado no boné, o dispositivo mede a inclinação da cabeça/tronco e identifica postura inadequada em tempo real, acionando um alerta sonoro para ajudar o usuário a manter alinhamento postural.
